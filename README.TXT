CHATOT.HTA
==========

Chatot for Windows. Opens a window that displays an image of Chatot.
The window moves around the screen.


System Requirements
-------------------

* Windows 98 or ME
* Internet Explorer 5 or 5.5


Running
-------

The files are AutoRun compatible. Copy the files onto a floppy disk or
burn to a writable CD.

If you are using a virtual machine, copy the files into an ISO or
access them in a SMB File Sharing folder. Navigate to the files in
Windows Explorer and double-click the file named Chatot (CHATOT.HTA)
with the Windows logo.
